Funnel template

Funnel template

Funnels are everywhere. They are among the easiest way to vizualise the number of users who completed a step. Add your events to create a basic funnels and count the number of times a specific event was triggered. Turn the "unique user" toggle on if you want to count by unique customers only 💪


You can use funnels for:
— Acquisition: Signup funnel
— Activation: Onboarding funnel, feature activation funnel
— Virality: Team invite funnel
— Monetization: Free-to-Paid funnel
— Testing: Fake door button funnel


💡Note: If you have more than one use case we'll soon allow you to duplicate this template.
